# College-Fest-Designs
## College Fest Design Project

In this project, I worked on designing multiple elements for a college fest to create a complete and consistent visual identity. My goal was to make the designs attractive, easy to understand, and suitable for students.

---

### Logo Design
I designed the logo for the college fest by focusing on simplicity and uniqueness. I chose colors and fonts that match the theme of the event and make the logo visually appealing. The design was kept clean so that it can be easily used on posters, ID cards, and certificates. I made sure the logo looks professional and attractive, creating a strong identity for the fest.


###  Main Poster
I created the main poster to grab attention and promote the event effectively. I used bold typography and proper spacing so that all important details are clearly visible. The design was made colorful and eye-catching to attract students.

### Food Pass
For the food pass, I designed a simple and clean layout so it is easy to read and use. I focused on clarity and proper alignment to make it practical as well as visually nice.



###  ID Card
I designed the ID card in a professional way by including necessary details and keeping the layout structured. The design is neat, easy to carry, and matches the fest theme.



###  Technical Event Poster
I created a separate poster for technical events with a modern and slightly futuristic look. I used design elements that represent technology and innovation.



###  Certificate
The certificate was designed with a formal and elegant style. I focused on alignment, spacing, and font selection to make it look official and presentable.



###  Website Poster
I designed a website promotion poster that highlights key information clearly. The layout is clean and informative so users can quickly understand the purpose.



###  Tools Used
- Canva  
- Figma  

---

###  Conclusion
Through this project, I learned how to maintain consistency across different designs and how to create visually appealing content for a specific audience.
